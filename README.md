# Distributed Data Query Application

This is the main repository for the Daquery application. It contains the following components:
 * ui/ - the web ui 
 * ws/ - the main web service
 * central/ - a central web service used to help during initial application setup.
 * common/ - common components used by both the main ws and the central web service

See additional README files in each component directory for more information about the component and build information.
